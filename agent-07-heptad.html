The Problem That Changed Mathematics

In 1736, Leonhard Euler solved the famous problem: Can you walk through Königsberg crossing each of its seven bridges exactly once?

Graph Representation:
$$G = (V, E)$$ $$V = \{A, B, C, D\} \quad \text{(four land masses)}$$ $$|E| = 7 \quad \text{(seven bridges)}$$
Degree Sequence:
$$\deg(A) = 5, \quad \deg(B) = 3, \quad \deg(C) = 3, \quad \deg(D) = 3$$ $$\text{All four vertices have odd degree!}$$

Euler's Theorem: A connected graph has an Eulerian path if and only if it has exactly 0 or 2 vertices of odd degree.

Application to Königsberg:
$$\text{Number of odd-degree vertices} = 4$$ $$4 \neq 0 \text{ and } 4 \neq 2$$ $$\therefore \text{ No Eulerian path exists}$$
