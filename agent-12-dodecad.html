IV. Modular Forms: Weight-12 Discriminant Δ(τ)

The Discriminant Function

The weight-12 cusp form Δ(τ) is the crown jewel of classical modular form theory, with deep connections to number theory, elliptic curves, and string theory.

Definition of Δ(τ)
Δ(τ) = (2π)¹² η(τ)²⁴ where η = Dedekind eta function
= q ∏(n=1 to ∞) (1-q^n)²⁴ where q = e^(2πiτ)
= ∑ τ(n)q^n = q - 24q² + 252q³ - 1472q⁴ + 4830q⁵ - ...
τ(n) = Ramanujan tau function

Ramanujan's Conjectures

Ramanujan (1916) conjectured remarkable properties of τ(n):

  • Multiplicativity: τ(mn) = τ(m)τ(n) when gcd(m,n)=1
  • Congruence: τ(p) ≡ 1 + p¹¹ (mod p) for prime p
  • Bound: |τ(p)| ≤ 2p^(11/2) for prime p

All three were eventually proven (Mordell 1917, Deligne 1974).

Example values:

τ(2) = -24
τ(3) = 252
τ(5) = 4830
τ(7) = 84480
